# ☕ Exercícios de Java (POO)

Repositório dedicado ao estudo e prática de **Programação Orientada a Objetos** com a linguagem Java. 

---

## 🚀 Tecnologias Utilizadas
* **Linguagem:** Java 25 
* **IDE:** Eclipse
* **Controle de Versão:** Git & GitHub

## 🛠️ Como rodar os projetos
1. Clone o repositório: `git clone https://github.com/CarolineOliveiraSilva/exercicios-java.git`
2. Importe a pasta do exercício desejado no seu Eclipse (File > Import > General > Existing Projects into Workspace).
3. Execute a classe `Program.java`.
