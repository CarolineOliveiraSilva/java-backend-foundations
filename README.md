# Trilha de Especialização Java (Back-end)

Repositório dedicado ao estudo contínuo e prático da linguagem Java. Este espaço documenta a minha evolução técnica, partindo dos fundamentos de lógica e manipulação de memória até a arquitetura de software e design de sistemas orientados a objetos.

---

## Tecnologias e Ferramentas

* **Linguagem:** Java 25
* **Ambiente de Desenvolvimento (IDE):** Eclipse / Spring Tools
* **Controle de Versão:** Git & GitHub

## Como executar localmente

1. Clone este repositório no seu terminal: 
   `git clone https://github.com/CarolineOliveiraSilva/exercicios-java.git`
2. Importe a pasta do repositório na sua IDE (No Eclipse: `File > Import > General > Existing Projects into Workspace`).
3. Navegue até o diretório `src` do exercício desejado.
4. Execute o arquivo `.java` principal da aplicação.
